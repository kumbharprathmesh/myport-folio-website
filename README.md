# Prathmesh - Personal Portfolio

A premium, interactive personal portfolio website built with modern web technologies to showcase my AI engineering projects, technical skills, and professional journey.

## 🚀 Features
- **Immersive Design:** Dark-themed, high-contrast aesthetic with ambient lighting and neon effects.
- **Scrolly-telling Experience:** Smooth scroll animations and 3D image flip sequences guided by user interaction.
- **Interactive Projects Grid:** Clean Bento-style grid layout focusing on critical 3D Animated Web experiences.
- **Dynamic Timeline:** Visual representation of my AI engineering career and educational background from a Tier 3 college.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Custom SVG & Lucide React

## 🏁 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm, yarn, or pnpm

### Installation

Clone the repository:
```bash
git clone https://github.com/kumbharprathmesh/myport-folio-website.git
cd myport-folio-website
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio locally.
